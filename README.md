# uuidgen
go implementation for uuidgen command

## install
via go 

```bash
go install github.com/mkusaka/uuidgen/blob/main/cmd@latest
```

via script
```bash
bash <(curl https://raw.githubusercontent.com/mkusaka/uuidgen/main/scripts/install.bash)
```

## usage

```bash
## install from go
uuidgen

## instlal from bash
./uuidgen
```
